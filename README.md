# Wonunger — пустой Node.js проект

Этот репозиторий — пустой scaffold Node.js приложения, подключающий пакет из GitHub: https://github.com/anfy1284/my-old-space

Как запустить:

1. Установите зависимости:

```bash
npm install
```

2. Запустите приложение:

```bash
npm start
```

Для разработки используйте:

```bash
npm run dev
```

Файл `src/index.js` пытается `require('my-old-space')` и вернёт поле `hasMyOldSpace`.
